Windows 软件推荐

Windows

整理 Windows 平台的常用软件推荐，涵盖安全防护、解压卸载、开发工具等分类的精选应用

标签：
软件推荐 安全软件 开发工具
发布于 2022年8月22日

安全软件 ​

1. 火绒 ​

火绒

专业尽职，极致低调，只为巩固安全防线，不打扰，是我们的态度,易使用，是我们的温度

2. 微软电脑管家 ​

微软电脑管家

简单无打扰，安全又安心，微软官方出品

解压软件 ​

1. Bandizip ​

Bandizip

Bandizip，是一个强大的多功能压缩文件管理工具，可提供很快的解压缩速度和其他各种实用功能。这是一个免费的软件，其付费版本更是可以使用更多的实用功能。

  • 新版有广告，影响不大

卸载软件 ​

1. GEEK UNINSTALLER ​

GEEK

Efficient and Fast, Small and Portable. 100% Free Clean Removal and Force Removal Native X64 support Easy-to-use User Interface Uninstall Windows Store Apps

  • 专业版收费

图像编辑器 ​

1. PicPick ​

PicPick

一款全能的设计工具，并且这款工具适宜每个人使用。一个全功能的屏幕截图工具,直观的图像编辑器,颜色选择器,颜色调色板,像素标尺,量角器,瞄准线和白板等等。

  • 收费

SSH连接工具 ​

1. Tabby ​

Tabby

Tabby (前身是 Terminus) 是一个可高度配置的终端模拟器和 SSH 或串口客户端，支持 Windows，macOS 和 Linux

2. FinalShell ​

FinalShell

FinalShell是一体化的的服务器,网络管理软件,不仅是ssh客户端,还是功能强大的开发,运维工具,充分满足开发,运维需求.

  • 收费开启所有功能

Deepl

2. 输入法状态跟踪 - ImTip ​

ImTip

全世界最精准的机器翻译.

3. Dns工具 - Dns Jumper ​

Dns Jumper

It can aid in accessing blocked websites, improve security, protect children by blocking inappropriate sites, speed browsing, and simplify DNS server management.
