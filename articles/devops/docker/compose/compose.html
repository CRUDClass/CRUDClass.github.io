Install Docker Compose plugin

Download and install Docker Compose on Linux with this step-by-step handbook. This plugin can be installed manually or by using a repository.

Docker Compose

Define and run multi-container applications with Docker - docker/compose: Define and run multi-container applications with Docker

环境 ​

  1. Linux版本：CentOS 7.9 64位操作系统
  2. Docker版本：19.03

下载并安装 Compose CLI 插件 ​

  1. 设置环境变量
  2. 创建cli——plugins文件夹
  3. 下载Docker Compose[^1]

[^1]: 因为是给root账号添加所以该文件下载位置在/root/.docker/cli-plugins/Docker Compose

root用户

bash
 DOCKER_CONFIG=${DOCKER_CONFIG:-$HOME/.docker}
 mkdir -p $DOCKER_CONFIG/cli-plugins
 curl -SL https://github.com/docker/compose/releases/download/v2.7.0/Docker Compose-linux-x86_64 -o $DOCKER_CONFIG/cli-plugins/Docker Compose

所有用户

bash
 mkdir -p /usr/local/lib/docker/cli-plugins
 curl -SL https://github.com/docker/compose/releases/download/v2.7.0/Docker Compose-linux-x86_64 -o /usr/local/lib/docker/cli-plugins/Docker Compose

对二进制文件应用可执行权限 ​

root用户

bash
chmod +x $DOCKER_CONFIG/cli-plugins/Docker Compose

所有用户

bash
sudo chmod +x /usr/local/lib/docker/cli-plugins/Docker Compose

测试安装 ​

bash
docker compose version

结果：Docker Compose version v2.7.0

到此安装成功

手动安装 ​

跳过下载文件步骤，创建/usr/local/lib/docker/cli-plugins/文件夹手动上传下载好的Docker Compose

Docker Compose下载位置

执行sudo chmod +x /usr/local/lib/docker/cli-plugins/Docker Compose

卸载 ​

删除二进制文件即可

root

bash
sudo rm /root/.docker/cli-plugins/Docker Compose

所以用户

bash
sudo rm /usr/local/lib/docker/cli-plugins/Docker Compose
