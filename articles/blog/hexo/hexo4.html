Hexo生成站点地图并提交到谷歌

Hexo

Hexo静态博客框架的生成站点地图并提交到谷歌，包括主题配置、文章管理和部署优化

标签：
SEO 站点地图 Google Search Console
发布于 2022年7月23日

准备工作 ​

登陆谷歌Google Search Console，根据网站提醒绑定你的博客域名。

安装插件 ​

进入hexo根目录执行如下命令

shell
npm install hexo-generator-sitemap --save

控制台输出 added 1 package in 2s
