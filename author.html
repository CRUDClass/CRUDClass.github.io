碎碎念 ​

这是一个普通开发者记录技术成长的博客，内容可能不够深入，但都是真实的实践经历。

起步：2018 年的探索 ​

最初的动力源于对 Linux 系统的好奇。为了深入学习，我购买了阿里云的轻量服务器，开始了"折腾"之旅：配置 IP、绑定域名、安装 Docker，这些基础操作让我对服务器管理有了初步认识。

Hexo 时代：博客初体验 ​

看到网上众多技术博客后，我也萌生了写博客的想法。经过调研，选择了 Hexo 作为博客框架，并采用了美观的 Fluid 主题。在学习过程中，参考了 Fluid's blog 的 利用 GitHub Actions 自动部署 Hexo 博客教程，不仅成功搭建了博客，还根据教程完善了内容展示和样式设计。

自动化部署：技术栈扩展 ​

有了服务器后，自然想实现自动化部署。这期间学习了：

  • Drone - 持续集成工具
  • Nginx - Web 服务器
  • Docker Compose - 容器编排

通过这些技术的组合，成功搭建了完整的自动化部署流程。

中断与重启：2025 年的回归 ​

随着工作忙碌和家庭琐事增多，博客更新逐渐停滞。直到 2025 年，重新燃起技术分享的热情，决定重启博客。

新的征程：VitePress 时代 ​

重启之初尝试了 VuePress，在学习 Vue 相关知识的过程中，发现了更轻量、更快速的 VitePress。VitePress 的简洁设计和优秀性能更符合我的需求，于是便迁移到了 VitePress，开启了新的博客之旅。
