Errors during downloading metadata for repository 'AppStream'

Errors during downloading metadata for repository ‘AppStream’

错误产生

  1. 阿里云服务器
  2. linux版本centOS8.2
  3. 执行 yum install -y [软件名]

错误内容

错误解决方法

  1. 进入阿里云开源镜像站-repo

  2. 下载：Centos-vault-8.5.2111.repo

  3. 进入文件夹

  4. 删除所有后缀为.repo的文件

  5. 上传Centos-vault-8.5.2111.repo

  6. 执行yum makecache
